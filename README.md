## Installation Requirements

* Node js
* yarn
* Git

### Installation code
run: 
```Bash
    $ git clone https://github.com/habib2013/heimdallinc.git
    $ cd heimdallinc
    $ yarn install
    $ yarn start
```

### Tests
```Bash
    $ yarn test
```
